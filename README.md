# 🌊 بحر 777 - الموسوعة العملاقة الذكية (Diamond+ Edition)

   ## 🚀 طريقة التنصيب السريعة (بدون تعديل):
   1. افتح تطبيق Termux (للأندرويد) أو أي استضافة تدعم بايثون.
   2. حدّث الحزم وثبّت المكتبات المطلوبة بنسخ هذا الأمر:
```bash
      pkg update && pkg upgrade -y
      pkg install python -y
      pip install telethon pytz Pillow requests
```
   3. شغّل السورس بالأمر التالي:
```bash
      python main.py
```
   4. أدخل رقم هاتفك وكود التفعيل عندما يطلب منك ذلك، وسيتم إنشاء الجلسة تلقائياً.
